Meme sound pack (myinstants.com clips picked by the project owner).

The MP3 files are NOT bundled in the repository. To fetch all of them,
run once on a machine with normal internet access:

    node scripts/download-sounds.mjs

Every clip lands here as <id>.mp3 (e.g. lepego.mp3). All files are
optional at runtime — the game stays silent for any missing clip.
You can also drop in your own custom clips using the same ids.
